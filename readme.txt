Adds a 'customer confirm email' field to the WooCommerce checkout page.

== Description ==

"WooCommerce Confirm Customer Email" will help you to add a field in Woocommerce checkout page. Before the final order customer or user will have to confirm their mail to complete the order. This plugin will protect your store from spam orders because the customer has to verify their mail before ordering from your store.

"WooCommerce Confirm Customer Email" is compatible with WooCommerce 3.0 or above versions.

Benefits of "WooCommerce Confirm Customer Email"

•	authenticate your buyer’s identity
•	Eliminate spam orders
•	Have supplementary information for chargeback

== Usage ==

Just upload plugin or add it from plugin section and it will work without any extra settings

== Installation ==

Install "WooCommerce Confirm Customer Email" by searching for "WooCommerce Confirm Customer Email" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
